Custom 3D Mesh Hand Brace Design

This project outlines the process of designing a custom 3D mesh pattern for a hand brace using Grasshopper, a visual programming language for Rhino 3D. The design focuses on creating a comfortable and functional product.

Importing and Mapping the Hand Mesh

Use MediaPipe to detect key hand joint points and output their coordinates, then import the 3D hand mesh as an STL or OBJ file into Rhino/Grasshopper.

Design Process Highlights

  • Map MediaPipe points onto the 3D hand mesh surface to follow the hand's curvature.
  • Apply Boolean operations to modify the mesh for comfort in key areas.
  • Create hexagonal grids and citron star-like mesh patterns with positive and negative movements controlled by sliders.
  • Use attractor points and smoothing algorithms to refine the shape and blend individual hexagonal faces into a natural organic surface.
  • Export finalized brace as STL or OBJ for 3D printing.

Key Design Considerations

  • Manufacturability: Designed for 3D printing constraints to ensure successful production.
  • Aesthetics: Inspired by automotive ornamentation and organic fruit forms.
  • Functionality: Provides support while allowing necessary movement.
  • Comfort: Prioritizes user comfort via material placement and thickness adjustments.

The custom brace design balances ergonomic support and an appealing form to encourage wearability and comfort.
